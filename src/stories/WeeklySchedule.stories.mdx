# WeeklySchedule

WeeklySchedule dialogs.

When requiring users to interact with the application,
but without jumping to a new page and interrupting the user's workflow,
you can use `WeeklySchedule` to create a new floating layer over the current page to get user feedback or display information.

`Props` are listed at the end of this page.

[View source](https://github.com/ahmedalatawi/react-beautiful-schedule/src/components/Schedule/WeeklySchedule)
